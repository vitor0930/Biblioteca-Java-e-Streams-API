# Sistema de gerenciamento de biblioteca
Sistema de gerenciamento de biblioteca feito em Java com foco em POO e estrutura de dados  

O sistema permite o cadastro de usuários e livros e o empréstimo e devolução de livros, associando livros a usuários

## Funcionalidades

- Cadastrar livros  
- Cadastrar usuários  
- Emprestar livros  
- Devolver livros  
- Listar livros (cadastrados, disponíveis e emprestados)  
- Listar usuários cadastrados  

## Tecnologias

- Java  
- POO  
- Streams API (filter, map, findAny, sorted e foreach)  
- Estruturas de dados (ArrayList e Map)  
- Collections

## Conceitos aplicados

- Encapsulamento  
- Manipulação de estruturas de dados  
- Objetos compostos  
- Organização em camadas

## Como executar

1. Clone o repositório
2. Compile o projeto
3. Execute a classe Main
